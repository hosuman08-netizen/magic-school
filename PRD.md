## Concept
**마법 아카데미 앱** — 플레이어가 마법 학교 학생이 되어 마법을 배우고, 연습하고, 성장하는 몰입형 학습 + 플레이 앱.

### Core Loop (ALWAYS LEARNING 중심)
1. **수업 (Lessons)**: 매일 다른 마법 주제. 간단한 연습 미니게임으로 "착수".
2. **연습 (Practice)**: Spell casting simulator. 성공/실패/near-miss로 variable reward.
3. **깨달음 (Insights)**: 연습 후 자동 "공부 모드" — 무엇을 배웠는지, 어떻게 이해했는지 기록 + 제안 (Legion doctrine 직적용).
4. **성장 (Progress)**: Magic Power, Knowledge, Spell Mastery. House ranking.
5. **소환 (Familiars)**: p3 AI companions를 "정령"이나 "동료 학생"으로 소환해 함께 연습 (cross-project).

### Theme
- 학교 이름: "Aether Academy" 또는 "아르카눔 마법 학원"
- Houses: Flame (공격), Veil (환영), Root (자연/방어), Void (금지 마법)
- 마법 = "배움의 힘". 진짜 스킬처럼 느껴지게 (암기, 타이밍, 패턴 인식).

## Features (MVP)
- Dashboard: 현재 레벨, 오늘의 수업, Energy (daily limit)
- Lessons: 4개 카테고리, 클릭으로 연습 시작
- Casting: 
  - Simple: 타이밍 게임 (click at right moment)
  - Memory: 주문 패턴 따라하기
  - Near-miss: 아슬아슬하게 성공하면 보너스 인사이트
- Spellbook: 배운 주문 목록 + 설명 + "깨달음" 노트
- Study Log: 매 세션 후 "이 과정에서 배운 점" 자동 생성 (ALWAYS LEARNING) + p6 Lung Surprise Eye + Ache-Breath cross DNA (voice echo from 창발 pain)
- FOMO: 
  - 한정 "월식의 밤" 이벤트 (희귀 주문)
  - Streak bonuses
  - "다른 학생들 지금 X명 연습 중" (social scarcity, fictional)
- Parallel: 여러 주문을 동시에 연습 모드 (병렬 업그레이드)

## Legion Doctrine Full Apply
- **무조건 과정 속 학습**: 모든 캐스팅 후 "배움 / 이해 / 깨달음" 3단계 질문 + 기록.
